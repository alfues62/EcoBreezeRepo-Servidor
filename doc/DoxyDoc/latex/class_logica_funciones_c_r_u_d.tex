\doxysubsection{Detailed Description}
Clase encargada de gestionar las operaciones CRUD (Crear, Leer, Actualizar, Eliminar) en la base de datos. 

Esta clase se encarga de manejar todas las operaciones que interactúan directamente con la base de datos de usuarios, mediciones y otros registros. Permite realizar la creación, lectura, actualización y eliminación de datos en la base de datos, gestionando las consultas SQL necesarias para modificar los datos persistentes.

Métodos principales\+:
\begin{DoxyEnumerate}
\item Crear nuevos registros de usuarios, mediciones y otros elementos en la base de datos.
\item Actualizar registros existentes con nueva información proporcionada.
\item Eliminar registros de la base de datos de acuerdo a ciertos parámetros.
\end{DoxyEnumerate}

\begin{DoxyNote}{Note}
Utiliza {\ttfamily PDO} para interactuar con la base de datos de manera segura y eficiente. 

Esta clase realiza modificaciones en los datos de la base de datos, por lo que puede tener efectos permanentes. 

Requiere un objeto de conexión a la base de datos previamente configurado. 
\end{DoxyNote}
